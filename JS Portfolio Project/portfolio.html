my projects

cyberpunk style vending machine


description

the purpose of this project was to create a user interface for a vending machine. items from the machine were to be obtained by calling to an api.

my apprach was to design the vending machine with a cyberpunk aesthetic.

goals

  • users should be able to click on a menu item and it should display in the message window
  • users should be able add money to the vending machine, it should display in the balance
  • users should be able to click purchase after selecting an item, displaying the result in the message window
  • users should be alerted if an item is out of stock or if purchase is attempted when balance is less than cost of item
  • after purchase, the user should receive their change and balance should be set to 0

technologies

  • html
  • css
  • ajax
  • jquery

library management system


description

the purpose of this project was to create an application that allowed librarians to make edits to books in a library, and grant access to patrons

this was a group project, utilizing an agile environment

goals

  • librarians and patrons should be able to login
  • librarians and patrons should be able to view all books in the library
  • librarians create/update/delete books and grant access to patrons
  • patrons should be able to checkout books if their account has been granted access
  • patrons should not be able to check out a book that is currently checked out by another patron
  • patrons should be able to return books

technologies

  • html
  • css
  • java
  • jsp servlets
  • tomcat
